# YOLOv8 Quickstart

This project demonstrates a basic object detection implementation using the Ultralytics YOLOv8 model.

## Task Objective
- Load a pre-trained YOLOv8 model.
- Perform object detection on an image.
- Save the results showing detected objects with bounding boxes.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python detect.py`
